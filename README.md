## Highlights

- **+54% aerodynamic efficiency** — improved L/D ratio from 6.5 to ~10 on a tandem-wing VTOL UAV through iterative CFD-driven redesign
- **35% drag reduction** — achieved through systematic aerodynamic optimisation of UAV configuration using ANSYS Fluent and OpenFOAM
- **Full multiphysics workflow** — linked aerodynamic loads → structural stress → thermal heat generation for system-level UAV performance assessment
- **Low-to-high fidelity validation** — consistently correlated XFLR5 predictions against 3D CFD results across operational Reynolds number ranges
